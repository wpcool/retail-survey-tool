# 零售调研小程序

微信小程序端的零售调研工具，用于调研员记录商品价格信息。

## 功能特性

- **任务管理**: 查看今日调研任务，了解任务详情
- **数据录入**: 便捷录入商品信息、价格、店铺等数据
- **定位功能**: 自动获取调研地点位置信息
- **个人中心**: 查看统计信息和个人资料

## 项目结构

```
mini-program/
├── app.js              # 应用入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── pages/
│   ├── login/          # 登录页
│   ├── index/          # 首页-任务列表
│   ├── create/         # 新建调研记录
│   └── profile/        # 个人中心
└── images/             # 图片资源
```

## 开发环境

- 微信开发者工具
- 基础库版本: 2.30.0+

## 快速开始

1. 使用微信开发者工具打开 `mini-program` 文件夹
2. 在 `app.js` 中修改 `baseUrl` 为实际后端地址
3. 开启"不校验合法域名"选项进行开发测试

## 注意事项

- 小程序需要配置后端API地址
- 定位功能需要用户授权
- 腾讯地图API需要申请Key
